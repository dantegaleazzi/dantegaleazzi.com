# Design System

## Direction

The launch visual direction is a single fixed system inspired by desktop documents, personal publishing and restrained brutalism.

There is no public theme selector.

## Palette

```css
--color-ink: #1e1e1e;
--color-paper: #f7f2e8;
--color-signal: #ffd400;
--color-orange: #ff5c1a;
```

- Warm dotted page background: `#f2ede4`
- Primary panels: white
- Primary accent and calls to action: yellow
- Orange is reserved for small secondary accents

## Typography

- Display and body: Space Grotesk
- Interface labels and metadata: DM Mono

Headlines are large, compact and strongly tracked. Labels use uppercase mono typography.

## UI language

- Black structural borders
- White content surfaces
- Restrained corner radius
- Small mechanical shadows
- Browser-window bars on document-like panels
- Yellow highlight behind important phrases
- No gradients
- No soft SaaS-style cards

## Section pattern

Primary content sections use:

1. Small uppercase mono kicker
2. Large editorial heading
3. Optional count aligned right
4. Content grid or interface block below

Current examples:

- Workflow inventory / Popular workflows
- Check the newsletter / Make AI Do The Work
- Tools / assets / useful links / Free Resources

## Responsive intent

- Desktop-first composition
- Sidebar and hero stack on mobile
- Resource cards switch from two columns to one
- Navigation remains compact
- The mobile profile includes a shortcut to the main subscribe form

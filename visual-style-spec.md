# Visual Style Specification

## Fixed launch style

The former `Desktop OS Y` experiment is now the only production style.

### Background

- Warm beige base
- Subtle radial dot pattern at `20px`

### Surfaces

- White header, profile and hero panels
- Black `2px` borders
- Rounded corners between `8px` and `12px` on major panels
- Resource rows remain square and compact

### Accent

- Primary yellow: `#FFD400`
- Used for:
  - hero underline
  - subscribe buttons
  - active filters
  - selected workflow card
  - hover states

### Typography

- Space Grotesk for editorial content
- DM Mono for navigation, labels, counters and badges

### Header

- Wordmark left
- Workflows, Newsletter, Free Resources and About Me right
- No theme selector

### Free Resources

- White two-column directory on desktop
- One column on mobile
- Compact rows with icon, name, description, category and external-link indicator
- Filter pills: All, Tools, Courses, Articles
- Active filter uses yellow background and `aria-pressed`

### Motion

- Short stepped underline animation in navigation
- Small mechanical hover shifts
- Motion disabled when `prefers-reduced-motion` is active
